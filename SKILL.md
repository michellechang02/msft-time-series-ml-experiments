# Time-Series ML Evaluation Skill

This skill enables an AI agent to run, evaluate, and summarize
a time-series ML ensemble experiment with rigorous evaluation.

## Capabilities
- Load historical MSFT price data
- Generate features with leakage prevention
- Train ensemble classifiers
- Evaluate out-of-sample performance with transaction costs
- Produce summary metrics and plots

## Usage
1. Run `time_series_ml_ensemble_learning.ipynb`
2. Execute all cells top-to-bottom
3. Extract:
   - Sharpe ratio
   - Max drawdown
   - Cost sensitivity results
4. Summarize findings and limitations
